# NovaTech Solutions - Frontend

Frontend del sistema de gestión desarrollado por **NovaTech Solutions** para el Proyecto Integrador del Instituto de Formación Técnica Superior N°29.

## Descripción

Este repositorio contiene la aplicación frontend del sistema Lavapro, desarrollada con React y Vite. La plataforma permite la gestión integral de una lavandería, incluyendo administración de usuarios, pedidos y operaciones del sistema.

## Acceso a la aplicación

La versión desplegada se encuentra disponible en:

**https://app.lavapro.online**

## Tecnologías utilizadas

* React
* Vite
* JavaScript
* CSS
* React Router
* Docker
* Nginx
